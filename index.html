About

UCRPC is ACM@UCR's entry-level programming contest. It is open to all students at UCR. Teams of exactly 3 people have 5 hours to solve 10 programming problems. The top-ranking teams win prizes. The best freshman team also wins a prize.

Home ← → Rules

Rules

  • You may use online resources during the contest
  • Each problem has a set of test cases and a runtime limit.
  • Grading scale is all or nothing; your program MUST pass all the test cases within the runtime limit in order to score points.
  • You cannot see test cases other than the provided sample case
  • Ranking is based on score, then on total submission time (each failed attempt counts as a time penalty)
  • You may use C++, C#, Java, or Python.
  • We provide the standard library for each language
  • A problem's runtime limit is fixed; we do not adjust for programming language.
